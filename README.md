# KNIME Data Automation Projects

Data automation and data preparation workflows built in KNIME, focused on transforming raw datasets into structured, analysis-ready outputs.

These workflows demonstrate how manual data processing tasks can be automated to improve efficiency, consistency, and data reliability.

## What This Project Shows
- Automated data ingestion from CSV sources
- Data cleaning and standardization processes
- Lookup table integration for classification
- Automated transformation logic
- Structured analytical output generation

## Tools & Technologies
- KNIME Analytics Platform
- Microsoft Excel

## Skills Demonstrated
- Data Analysis
- Data Cleaning & Preparation
- Workflow Automation
- Data Transformation
- Process Optimization

## Data Disclaimer
Due to confidentiality restrictions, original datasets are not included.  
This repository focuses on workflow design, automation logic, and data processing structure.
